**Currently in beta**

Phantom Editor is the iOS app for your [Ghost](https://ghost.org) blog.

### Features

- Markdown editing and post settings
- Works with **[Ghost Pro](https://ghost.org/pricing/)** (ghost.io) as well as **self-hosted blogs**
- Manage tags

### Dev Setup

You need a computer running MacOS and XCode 9 or above with iOS 11 simulators

- Clone the repo
- Run `$ carthage bootstrap` to install dependecies. [See Carthage docs](https://github.com/Carthage/Carthage)
- Open `Phantom.xcodeproj`
- Select `PhantomDev` scheme and Run (`Ctrl + R`)
